# 指尖三国：谋定天下

一个单文件纯前端塔防小游戏。玩家在固定行军路线上布置武将防线，利用军费、谋略值和主动技能挡住一波又一波敌军，完成守城通关。

## 在线体验

- GitHub 仓库：[yuyangjungle/1](https://github.com/yuyangjungle/1)
- GitHub Pages：`https://yuyangjungle.github.io/1/`

说明：仓库代码已经上传完成。若 Pages 还未开启，上面的体验链接暂时会返回 `404`。在仓库里打开 `Settings -> Pages`，将发布源设置为 `main` 分支的 `/(root)` 后即可生效。

## 项目亮点

- 单文件静态部署，无需后端，适合直接托管到 GitHub Pages
- `Canvas` 驱动的塔防主循环、寻路推进、投射物碰撞和波次刷新
- 四类武将防御塔，包含差异化射程、伤害、升级成长和主动技能
- 本地“军师建议”系统，会根据波次、资产、兵力和敌军压力给出提示
- `localStorage` 本地存档，可直接保存和恢复游戏进度
- 主题切换、Boss 波次、连斩奖励、粒子特效和简单 `Web Audio` 音效

## 玩法说明

1. 进入游戏后选择剧情模式或无尽模式。
2. 底部点击武将卡牌，再在地图空地上布阵。
3. 敌军沿固定路线前进，漏怪会扣减城池生命。
4. 击杀敌军可获得军费和谋略值。
5. 谋略值足够时，点击已部署的武将可释放主动技能。
6. 守住 10 波并击败 Boss 可完成剧情模式通关。

## 操作方式

- 鼠标点击底部卡牌：选择要部署的武将
- 鼠标点击地图空地：放置武将
- 鼠标点击已部署武将：释放主动技能
- 右侧工具栏：
  - `智囊妙计`：查看当前局势建议
  - `切换地貌`：切换地图主题
  - `本地存档`：保存当前进度
  - `读取进度`：恢复上次存档
  - `主菜单`：回到模式选择

## 技术实现

- `HTML`：页面结构与 UI 容器
- `CSS`：界面样式、按钮状态、覆盖层与布局
- `JavaScript`：核心游戏逻辑、实体系统、波次控制、技能与存档
- `Canvas 2D`：战场渲染、投射物、特效、范围预览
- `Web Audio API`：轻量即时音效
- `localStorage`：浏览器本地存档

## 项目结构

当前版本是单文件静态项目：

```text
.
├── index.html
└── README.md
```

之所以只有一个 `index.html`，是因为样式和脚本都内联在同一个文件里，部署门槛最低，适合快速演示和面试展示。

## 本地运行

直接用浏览器打开 `index.html` 即可运行。

如果想用本地静态服务器，也可以在当前目录执行任意一种方式：

```bash
python -m http.server 8000
```

或

```bash
npx serve .
```

## 部署说明

本项目适合部署到 GitHub Pages：

1. 将代码推送到 GitHub 仓库
2. 打开 `Settings -> Pages`
3. 在 `Build and deployment` 中选择 `Deploy from a branch`
4. 选择 `main` 分支和 `/(root)`
5. 保存后等待 GitHub Pages 发布

默认访问地址通常为：

```text
https://<github-username>.github.io/<repo-name>/
```

当前仓库名为 `1`，因此地址会是：

```text
https://yuyangjungle.github.io/1/
```

## 面试讲解建议

面试时可以重点讲这几个点：

- 为什么选择单文件静态架构：部署快，展示门槛低
- 如何组织游戏循环：敌军、塔、防御技能、投射物、粒子效果分层更新
- 修复过哪些问题：失败流程、技能击杀结算、静态部署兼容、读档恢复
- 如何继续扩展：更多兵种、地图、数值平衡、移动端适配、资源拆分
